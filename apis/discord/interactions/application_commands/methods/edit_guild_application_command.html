Edit Guild Application Command¶

ORCAcode discord_edit_guild_application_command(struct discord *client, const u64_snowflake_t application_id, const u64_snowflake_t guild_id, const u64_snowflake_t command_id, struct discord_edit_guild_application_command_params *params, struct discord_application_command *p_app_cmd)¶

PATCH /applications/{application.id}/guilds/{guild.id}/commands/{command.id}

Edit a guild command. Updates for guild commands will be available immediately. Returns an application command object.

See

https://discord.com/developers/docs/interactions/application-commands#edit-guild-application-command

Parameters

  • client – the client created with discord_init()

  • application_id – the unique id of the parent application

  • guild_id – the guild where the command is located

  • command_id – the registered command id

  • params – request parameters

  • p_app_cmd – the application command object if successful

Returns

ORCAcode for how the transfer went, ORCA_OK means a successful request
