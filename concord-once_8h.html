◆ ccord_has_sigint

int ccord_has_sigint
extern

If SIGINT is detected client(s) will be disconnected from their on-going session.

This global shall be set if a SIGINT is detected, running clients will then attempt to perform a clean disconnect, rather then just letting the program end abruptly.

Note
client shall only attempt to disconnect if there aren't any active events waiting to be listened or reacted to
